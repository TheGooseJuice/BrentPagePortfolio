Here's some of my Experience.

On my own I've created a Time Attack Racer, Tic Tac Toe, 2D Platformer, and a 3D Alien Ship Collect-a-thon. In a team we created "The Lost Box", a voxile 3d dungeon crawler with 4 classes, many monsters and traps! Using Blueprint and C++

Working with Phaser has been a great experience for me in terms of learning how to create a game start to finish. I refined my skills in sprite creation, physics, collisions, particle effects, UI design and development, score and high score tracking, loader and much more using this engine.

On my own I've created a Endless Runner, Car Soccer, 2D Platformer, and a 3D Pie. In a team we created "The Spot", a virtual Arcade where there is 5 games to choose from. I was the team lead for Air Hockey where I learned to network with Unity using C#
