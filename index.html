Here's some of my Experience.

As a solo project, I created an endless runner game, a car soccer game, 2D Platformer, and a 3D game titled "Pie Hop". In a team project, we created "The Spot", a virtual Arcade where there the player can choose from 5 mini games. Within the project the mini game I developed, as the team lead, was an air hockey game called "Tron Lazers". During this project I learned to network with Unity using C#

Working with Phaser has been a great experience for me in terms of learning how to create a game start to finish. I refined my skills in sprite creation, physics, collisions, particle effects, UI design and development, score and high score tracking, loader and much more using this engine.

On my own I've created a Time Attack Racer, Tic Tac Toe, 2D Platformer, and a 3D Alien Ship Collect-a-thon. In a team, we created "The Lost Box", a 3D dungeon crawler with 4 classes, many monsters and traps, using Blueprint and C++
